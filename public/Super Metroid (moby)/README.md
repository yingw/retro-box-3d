## 3D 盒子尺寸

**卡带方向**: 横向 (w > h)

**目标比例**: w:h:d = 1158:800:195 ≈ 5.9:4:1

**归一化尺寸** (除以 400):
- w = 1158/400 ≈ 2.9
- h = 800/400 = 2.0
- d = 195/400 ≈ 0.49
